## My path

I first came across Linux in 1998, when I was a student. It wasn't an easy start: as a poor student, I only had one computer. At weekends, I would try to install Linux on it. The installation was successful, but the Linux loader LILO only displayed `li` on an otherwise empty screen when trying to boot the computer.

By Sunday evenings, I would have to revert to an operating system that would enable me to complete my schoolwork for the following week.

Weekend after weekend, I continued to make progress, and I'm still trying to improve my Linux skills to this day. What fascinates me most is that even after thirty years, I still find the operating system's versatility and power intriguing and valuable.

If I haven't been able to achieve something, it's been down to me, and that has motivated me to learn more. Linux has never failed me.
