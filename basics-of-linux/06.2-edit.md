# Edit

## Nano

Nano is an easy-to-use text editor that can be used in a terminal window. To open an empty editor view, enter the command `nano`.

![Nano text editor](/imgs/nano.png)

You can then enter text and move around using the arrow keys, as well as a set of commands. Most of these commands use the control (`^`) key as a modifier.

a) Type in some text, for example, `Hello World`.
b) Press Ctrl+O. Enter the name `nano-hello` for the file
c) and accept the name with Enter.
d) Exit nano with Ctrl+X.

Enter the command `ls -l` and observe a file named `nano-hello` in the listing.

Try the commands `cat nano-hello` and `file nano-hello` to examine the file created by the nano editor.

## vim

Open the file from the previous task using the command `vim nano-hello`.

::: terminal
Hello World
~
~
~
~
"nano-hello" 1L, 12B                          1,1             All
:::

Vim has three modes: Normal mode, Edit mode, and Command-line mode. Edit mode could be categorised further, but for simplicity's sake, we will omit that for now.

Vim launches in normal mode.

a) Press the `w` (as in 'word') key.
b) Then press the `dw` (as in 'delete' and 'word').
c) Next, press the `a` ('append') key. _enters the edit mode_
d) Type in the word `vim` and then
e) press the `<ESC>` key. _returns the normal mode_
f) Lastly, type out `:wq` ('write' and 'quit') key events. _command-line mode_

Rename the file `nano-hello` to `vim-hello`. Then enter the command `cat vim-hello`.

::: terminal
{{HOSTNAME}}@{{USERNAME}}:~$ vim nano-hello
{{HOSTNAME}}@{{USERNAME}}:~$ mv nano-hello vim-hello
{{HOSTNAME}}@{{USERNAME}}:~$ cat vim-hello
Hello vim
{{HOSTNAME}}@{{USERNAME}}:~$ █
:::

If you wish, you can now copy the `vim-hello` file to `nano-hello`, open it with the `nano nano-hello` command and change the word 'vim' to 'nano'.
